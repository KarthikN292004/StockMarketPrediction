# StockMarketPrediction
Stock market prediction app using Facebook Prophet to forecast future prices from historical data. Features data preprocessing, trend and seasonality analysis, model evaluation (MAE, RMSE), and an interactive Streamlit dashboard for visualization.

# Overview

This project is a Stock Market Forecasting Web Application that predicts future stock prices using time series analysis. It leverages Facebook Prophet, a powerful forecasting model, to analyze historical stock data and generate accurate predictions.

The application also provides an interactive dashboard built with Streamlit, enabling users to visualize trends, seasonality, and future price movements.

# Features

Fetch real-time historical stock data using yfinance

Data preprocessing and cleaning using pandas

Time series forecasting using Facebook Prophet

Feature engineering (trend & seasonality analysis)

Model evaluation using MAE & RMSE

Interactive dashboard using Streamlit

Deployment support (Heroku / Streamlit Cloud)

# Tech Stack

Programming Language: Python

Libraries: Pandas, NumPy, Prophet, Matplotlib

Visualization: Streamlit

Data Source: yfinance API

Deployment: Heroku / Streamlit Cloud
